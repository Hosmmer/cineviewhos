# i18n — react-intl

## Flujo obligatorio

Cada vez que se agrega, cambia o elimina un texto visible en la UI:

1. Elegir o crear la clave (`SECTION.LABEL`, `PAGE.TITLE`, etc.)
2. Actualizar **ambos** archivos JSON: `src/i18n/es.json` y `src/i18n/en.json`
3. Usar la clave en el componente con `intl.formatMessage`

**Nunca** usar una clave en un componente sin haberla definido en los dos JSON.
**Al eliminar** un elemento de UI: borrar también su clave de los dos JSON.

## Ejemplo

```json
// es.json
{ "NAV.HOME": "Inicio", "NAV.ITEMS": "Elementos" }

// en.json
{ "NAV.HOME": "Home", "NAV.ITEMS": "Items" }
```

```tsx
import { useIntl } from 'react-intl'

const intl = useIntl()
title={intl.formatMessage({ id: 'NAV.HOME', defaultMessage: 'Home' })}
```

## Ubicación

```
src/i18n/
  es.json   ← español (idioma principal)
  en.json   ← inglés
```
